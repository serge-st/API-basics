API Docs Example

  • Supported methods: GET, POST, DELETE
  • Request base URL: http://localhost
  • Request bodies should be encoded using JSON format.
  • Responses use standard HTTP response codes to indicate errors
Desciption Path Method Request Response
Get All Users /users GET x Array of User Objects
Get 1 User /users/:id GET x User Object
Add a New User /users POST User Object x
Delete a User /users/:id DELETE x x

User Object Parameters:

Name Type Required/Optional
id number auto-generated
name string required
age number required
isAdmin boolean required
skills array required
languages object required
hobbies array optional
